EcoTracker

EcoTracker is a web-based application designed to help users track and manage environmentally friendly activities. The platform provides an easy-to-use interface for monitoring sustainable habits and promoting eco-conscious living.

Features

- User Registration and Login
- Dashboard for tracking activities
- Add, update, and delete records
- Responsive user interface
- Activity monitoring and reporting
- Easy navigation and user-friendly design

Technologies Used

Frontend

- HTML
- CSS
- JavaScript

Backend

- PHP (Planned)

Database

- MySQL (Planned)

Installation

1. Clone the repository:
   git clone https://github.com/Reshma2404/EcoTracker.git

2. Open the project folder.

3. Run the project using a local server such as XAMPP or VS Code Live Server.
